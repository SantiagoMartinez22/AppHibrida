# VigiLog

Aplicación híbrida (SPA) para digitalizar el registro de vigilancia en propiedad horizontal.

El objetivo es reemplazar el libro físico por un flujo de ingreso, registro de visitantes, consulta y entrega de turno.

## Stack

- React + TypeScript + Vite
- Tailwind + SCSS
- Zustand + localStorage
- React Router

## Funcionalidades actuales

- Inicio de sesión por rol (vigilante / admin) con usuario y PIN.
- Rutas separadas por rol.
- Registro de visitante.
- Listado y detalle de visitantes.
- Entrega de turno desde dashboard de vigilante.
- Historial reciente de entrega de turnos en dashboard admin.
