# VigiLog - Documentación pública

Esta carpeta se publica en **GitHub Pages** para proporcionar la URL de la política de privacidad requerida por Google Play Store.

## Habilitar GitHub Pages

1. Sube este repositorio a GitHub (si aún no lo has hecho).
2. Ve a **Settings** → **Pages** del repositorio.
3. En **Source**, selecciona **Deploy from a branch**.
4. En **Branch**, elige `main` (o la rama principal).
5. En **Folder**, selecciona **/docs**.
6. Guarda con **Save**.

## URL de la política de privacidad

Una vez desplegado, la URL será:

```
https://<tu-usuario>.github.io/<nombre-repo>/privacy.html
```

Ejemplo: `https://upb-edu-co.github.io/vigilog/privacy.html`

Usa esta URL en la ficha de la app en Google Play Console (Política de privacidad).
